## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo

```sh
git clone https://github.com/amanjagdev/salyd.git
```

2. Install packages

```sh
npm install | yarn 
```

3. Install Expo Dependencies

```sh
expo install
```
